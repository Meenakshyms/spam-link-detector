# Spam Link Detector

A web-based tool to detect potentially unsafe, phishing, or spam URLs using Python and Flask.

## Features
- Checks for unsecure HTTP links
- Detects unusually long domain names
- Identifies suspicious patterns in URLs
- Provides a professional, trustworthy report

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Regex
- tldextract

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Run the application  
   `python app.py`

3. Open browser at `http://localhost:5000`

## Use Case
Helps users quickly verify the safety of links before visiting, reducing phishing and spam risks.
